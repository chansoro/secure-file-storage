Secure File Vault

A secure, Flask-based file storage system with client-side encryption.

Features

Client-Side Encryption: Uses Web Crypto API (AES-GCM) to encrypt files before upload.

Secure Backend: Flask API handling JWT authentication.

No Database Config: Runs on SQLite automatically.

How to Run Locally

Install dependencies: pip install -r requirements.txt

Run server: python app.py

Visit http://localhost:5000

Deployment

Hosted on PythonAnywhere.
